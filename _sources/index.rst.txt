LittleDarwin: A Java Mutation Analysis Framework
=================================================

**LittleDarwin** is a mutation analysis framework for Java. It can be used to
generate mutants of your Java source code and then run a build command to see
if your test suite can detect the mutants.

This documentation will help you get started with LittleDarwin.
